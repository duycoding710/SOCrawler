# SOCrawler
Greetings! 
 
Want to know how to make script like this? Visit me at pynist.blogspot.com! 

##What does it do?
Display N top questions as well as the highest upvoted answer based on a given tag.

## How to excute?
```
chmod +x socrawler.py
./socrawler.py <N> <TAG>
```
